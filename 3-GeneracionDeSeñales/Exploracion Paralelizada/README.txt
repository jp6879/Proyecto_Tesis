Esta carpeta contiene un programa para la exploración de hiperparámetros de una NODE para predecir señales de Hahn y el codigo para enviarlo al cluster
Hay dos exploraciones realizadas, una realizada para 100 señales y menos datos muestreados y otra para 50 señales y mas datos muestreados.
Contiene ademas el análisis de las arquitecturas que mejor resultados otorgan.
